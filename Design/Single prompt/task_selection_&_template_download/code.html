Update Customer Sales-Rep

This process reassigns customers from one sales representative to another in bulk. Please follow the steps below carefully to ensure a successful update.

description

Instructions

1

Download Template: Click the button below to get the official Excel template. Only this exact format will be accepted for upload.

2

Fill Template: Open the file and enter the required data, such as Customer ID and the New Sales-Rep ID for each reassignment.

3

Upload File: Once your data is ready, return to this portal and use the 'Proceed to Upload' button to submit your completed file.

info

Please do not alter the column structure of the template file.
